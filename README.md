# STORE MANAGEMENT DESKTOP APPLICATION 🏪

An end-of-training project built with C# and SQL Server. The application streamlines daily supermarket operations, covering inventory management, order processing, employee management, and data reporting through an intuitive Windows Forms interface.

## FEATURES ✨

- **Inventory Management** — Track product stock levels with automatic updates triggered on each order.
- **Order Processing** — Create and manage customer orders with automatic total price calculation.
- **Customer Management** — Maintain a complete customer database with search and filtering.
- **Employee Management** — Manage employee records and handle role-based login access.
- **Data Export** — Export tables to XML, XSD, HTML, and Excel formats.
- **Charts & Reporting** — Visualize sales and stock data through built-in charts.
- **Dark Mode** — Toggle between light and dark themes across all application windows.

## TECHNOLOGIES 🚀

- **C#** — Programming language used for application development.
- **SQL Server** — Relational database management system for storing application data.
- **Windows Forms** — UI framework for building the desktop interface.
- **Visual Studio** — Integrated development environment used for coding and building the application.

## INSTALLATION ⚙️

> This guide assumes you have Visual Studio and SQL Server installed.

1. Clone the repository and open `Gestion d'un Supermarché.sln` in Visual Studio.
2. Import `Gestion d'un Supermarché.sql` into SQL Server to create the database schema and load sample data.
3. Update the connection string in `Outils.cs` to point to your SQL Server instance.
4. Build and run the application from Visual Studio.
